# GEPRCF722 2025.12.5 skipped CLI errors

These errors from `obj/Errors.txt` were not included in `GEPRCF722_2025.12.5_cli_fix.txt`.

- LED and color commands were skipped as requested.
- `resource LED_STRIP 1 A08` was skipped with the LED changes.
- `resource SERIAL_TX/RX 7..10 NONE` was skipped because this target only accepts serial indexes 1 through 6.
- `resource SOFTSERIAL_TX/RX 1..2 NONE` was skipped because those resource names are not valid in this build.
- `feature -RX_MSP` was skipped because `RX_MSP` is no longer a valid feature name here.
- `beeper RC_SMOOTHING_INIT_FAIL` was skipped because that beeper name is gone.
- `blackbox_disable_gps`, `gps_*`, and `gps_rescue_*` were skipped because the current GEPRCF722 build does not expose those GPS CLI settings.
- `min_throttle` was skipped because it is an old PWM-era setting and is not used for DSHOT motor idle.
- `osd_canvas_width` and `osd_canvas_height` were skipped because canvas size is not a CLI setting in this build.
- `gyro_1_sensor_align`, `gyro_1_align_*`, `gyro_2_sensor_align`, and `gyro_2_align_*` were skipped because the old per-gyro alignment CLI settings are gone. Use the board target defaults unless the target config itself needs changing.
- `d_min_roll`, `d_min_pitch`, and `d_min_yaw` were skipped because the old D_MIN settings are gone. Tune with the current D Max and simplified tuning controls instead.
